Schnellstart

In 15 Minuten loslegen

Klone das Repository und kopiere die ersten Templates direkt in dein Projekt.

# 1. Repository klonen
git clone https://github.com/luongnv89/claude-howto.git
cd claude-howto

# 2. Ersten Slash Command kopieren
mkdir -p /pfad/zu/projekt/.claude/commands
cp 01-slash-commands/optimize.md /pfad/zu/projekt/.claude/commands/

# 3. In Claude Code ausprobieren: /optimize

# 4. Projekt-Memory einrichten
cp 02-memory/project-CLAUDE.md /pfad/zu/projekt/CLAUDE.md

SLASH COMMANDS

Als .md-Dateien in .claude/commands/ ablegen. Mit /befehl aufrufen.

MEMORY

CLAUDE.md im Projektroot oder Unterordnern. Wird automatisch geladen.

SKILLS

In ~/.claude/skills/ oder .claude/skills/ im Projekt ablegen.

SUBAGENTEN

Als .md-Dateien in .claude/agents/. Werden automatisch delegiert.
